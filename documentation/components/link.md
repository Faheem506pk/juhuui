## Link

Another way of writing a tag.

#### Import component

```js
import { Link } from 'juhuui';
```

#### How to

```js-live
<Link
  color="blue.700"
  fontSize="xl"
  fun
  href="https://teuberkohlhoff.com/"
  _hover={{
    color: "blue.400"
  }}
>
  Sharing my love for Teuber Kohlhoff who designed this website
</Link>
```
